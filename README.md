# MERN-Admin-Panel

## Backend Node

``` bash

copy .env.example to .env
update MONGO_DB_NAME and JWT_SECRET on .env file

# install dependencies
npm install

# serve with hot reload at localhost:3000
npm run dev

```

## Frontend React

``` bash

copy .env.example to .env

yarn install

# run at localhost:8000
yarn start

# build for production with minification
yarn build

```
